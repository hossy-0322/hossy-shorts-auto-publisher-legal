# Hossy Shorts Auto Publisher Legal Pages

- [Terms of Service](terms.html)

## GitHub Pages

Use GitHub Pages with the following settings:

- Source: Deploy from a branch
- Branch: `main`
- Folder: `/ (root)`

Expected public URLs after Pages is enabled:

- `https://hossy-0322.github.io/hossy-shorts-auto-publisher-legal/terms.html`
